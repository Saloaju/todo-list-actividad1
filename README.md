# To Do List - Actividad para la Semana 1 o Unidad 1

Proyecto para llevar el control de tareas y metas personales mediante una aplicación web responsive construida con React Bootstrap y CSS.

## Instrucciones de ejecución
Este proyecto fue desarrollado en el entorno web Playcode.io, la forma de visualizarlo es:
**Visualización directa:**
1. Acceder al enlace del proyecto en Playcode proporcionado en la entrega.
2. El entorno instalará automáticamente las dependencias necesarias (`bootstrap` y `react-bootstrap`) y renderizará la aplicación.
**Ejecución en entorno local:**
1. Clonar este repositorio usando: `git clone <enlace-del-repositorio>`
2. Crear un entorno de React (Ejemplo: `npm create vite@latest`) e intercambiar la carpeta `src`.
3. Instalar dependencias: `npm install react-bootstrap bootstrap`
4. Correr el entorno de desarrollo: `npm run dev`
